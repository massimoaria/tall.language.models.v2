Pre-Trained Annotation Models for TALL (v2 — UDPipe 2)
================

## Overview

**TALL - *Text Analysis for ALL*** (<https://github.com/massimoaria/tall>) is an
application that provides a comprehensive suite of methodologies for various
text analysis tasks, designed to meet the needs of researchers who may lack
extensive programming skills.

This repository, `tall.language.models.v2`, provides the **second generation**
of pre-trained NLP annotation models for TALL: **UDPipe 2** models trained on
gold standard annotated corpora from the
[Universal Dependencies](https://universaldependencies.org/) (UD) project,
**version 2.17** (Straka, 2025 — source:
[LINDAT/CLARIAH-CZ](http://hdl.handle.net/11234/1-6046)). Compared to the
UDPipe 1 models of the first-generation repository
([`tall.language.models`](https://github.com/massimoaria/tall.language.models)),
UDPipe 2 models offer substantially higher tagging and parsing accuracy
(e.g. English LAS 78→92, Italian LAS 84→94).

The repository also hosts:

- the shared **mBERT** TensorFlow checkpoint
  ([google-bert/bert-base-multilingual-uncased](https://huggingface.co/google-bert/bert-base-multilingual-uncased),
  Apache-2.0) required by UDPipe 2 and reused by TALL for contextual word
  embeddings;
- **runtime data assets** (release `data-v1`): sentiment/emotion lexicons per
  language and keyness reference wordlists (OpenSubtitles-derived frequency
  lists), converted to CSV from the first-generation repository.

## Purpose

- To provide updated pre-trained **UDPipe 2** models for a wide range of
  languages, aligned with UD corpora version 2.17.
- To distribute each model as an individual GitHub Release asset (one
  `tar.gz` per model + a `models.json` manifest with sha256, size and
  per-treebank accuracy), so the TALL desktop app can download on demand with
  checksum verification — models are never bundled in installers.
- To facilitate reproducible NLP research by making these models and their
  evaluation metrics readily available.

## Data Source

The gold standard annotated corpora used to train these models are sourced
from the Universal Dependencies project
([Universal Dependencies GitHub](https://github.com/UniversalDependencies)),
version 2.17. Models were trained and evaluated by Milan Straka (ÚFAL MFF,
Charles University) and are redistributed here in per-model archives.

**License: CC BY-NC-SA 4.0** (non-commercial, share-alike) for the UDPipe 2
models, as inherited from the UD treebanks. mBERT: Apache-2.0. Lexicons and
wordlists (release `data-v1`): same provenance and licensing as the
first-generation repository (Hu-Liu / NRC-derived lexicons for research use;
OpenSubtitles frequency data).
